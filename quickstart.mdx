## Get an API key

Generate your API key by calling the keys endpoint:

```bash
curl -X POST https://api.shield-api.com/v1/keys \
  -H "Content-Type: application/json" \
  -d '{"customer_id":"your-company","name":"production","tier":"starter"}'
```

Response:
```json
{
  "key": "shld_..."
}
```

Save this key — you won't see it again.

## Moderate text

```bash
curl -X POST https://api.shield-api.com/v1/moderate \
  -H "Authorization: Bearer shld_..." \
  -H "Content-Type: application/json" \
  -d '{"text":"your content here"}'
```

Response:
```json
{
  "request_id": "4fa1705ea02a64a09548bf3257ee3f94",
  "decision": "block",
  "confidence": 0.91,
  "category": "toxic",
  "reasoning": "ml model detected toxic content"
}
```

## Use the Python SDK

```bash
pip install shield-moderation
```

```python
from shield_moderation import ShieldClient

client = ShieldClient("shld_...")

result = client.moderate_text("your content here")

if result["decision"] == "block":
    print(f"Blocked: {result['category']} ({result['confidence']:.0%} confident)")
else:
    print("Content is safe")
```

## Use the Node.js SDK

```bash
npm install shield-moderation
```

```javascript
const { ShieldClient } = require('shield-moderation')

const client = new ShieldClient('shld_...')

const result = await client.moderateText('your content here')

if (result.decision === 'block') {
  console.log(`Blocked: ${result.category}`)
}
```

## Use context signals

Shield can adjust decisions based on platform context:

```python
result = client.moderate_text(
    "breaking news: violent attack downtown",
    platform_type="news",    # news sites get more lenient treatment
    user_tier="trusted"      # verified users get more lenient treatment
)
```
